    Пространство имен System.Drawing

    .NET Framework (current version)
    Другие версии
    • Visual Studio 2010
    • .NET Framework 4
    • Visual Studio 2008
    • .NET Framework 3.5
     

    Опубликовано: Октябрь 2016

    Пространство имен System.Drawing обеспечивает доступ к базовым функциональным возможностям графического интерфейса GDI+. Пространства имен System.Drawing.Drawing2D, System.Drawing.Imaging и System.Drawing.Text обеспечивают дополнительные функциональные возможности.

    Классы

    КлассОписание
    Bitmap

    Инкапсулирует GDI+ точечного рисунка, который состоит из данных о пикселях графического изображения и его атрибуты. A Bitmap — объект, используемый для работы с изображениями, определяемых данными о пикселях.

    BitmapSuffixInSameAssemblyAttribute

    Указывает, что, когда Интерпретация ToolboxBitmapAttribute объявления, сборка будет искать указанных ресурсов в той же сборке, но с BitmapSuffix значение конфигурации, добавленным к имени файла объявленный.

    BitmapSuffixInSatelliteAssemblyAttribute

    Указывает, что, когда Интерпретация ToolboxBitmapAttribute объявления, сборка должна выглядеть для указанного ресурса во вспомогательную сборку, но с BitmapSuffix значение конфигурации, добавленным к имени файла объявленный.

    Brush

    Определяет объекты, используемые для заливки внутри графических фигур, таких как прямоугольники, эллипсы, круги, многоугольники и пути.

    Brushes

    Кисти для всех стандартных цветов. Этот класс не наследуется.

    BufferedGraphics

    Предоставляет графический буфер для двойной буферизации.

    BufferedGraphicsContext

    Предоставляет методы для создания графических буферов, которые могут использоваться для двойной буферизации.

    BufferedGraphicsManager

    Предоставляет доступ к объекту контекста основной буферизованной графики для домена приложения.

    ColorConverter

    Преобразует цвета из одного типа данных в другой. Доступа к этому классу через TypeDescriptor.

    ColorTranslator

    Преобразует цвета в и из GDI + Color структуры. Этот класс не наследуется.

    Font

    Определяет конкретный формат текста, включая начертание шрифта, его размер и атрибуты стиля. Этот класс не наследуется.

    FontConverter

    Преобразует Font объектов из одного типа в другой.

    FontFamily

    Определяет группу гарнитур шрифта со схожим базовым макетом и определенными различиями в стиле. Этот класс не наследуется.

    FontConverter.FontNameConverter

    Этот API поддерживает инфраструктуру продукт, и его не следует использовать напрямую из кода. FontConverter.FontNameConverter Представляет преобразователь типов, который используется для преобразования имени шрифта в другие представления и обратно.

    FontConverter.FontUnitConverter

    Этот API поддерживает инфраструктуру продукт, и его не следует использовать напрямую из кода. Преобразует единицах измерения и другие типы.

    Graphics

    Инкапсулирует поверхность рисования GDI+. Этот класс не наследуется.

    Icon

    Представляет значок Windows, который является точечным рисунком небольшого размера и используется для представления объекта. Значки можно считать прозрачными точечными рисунками, хотя их размер определяется системой.

    IconConverter

    Преобразует Icon объект из одного типа данных в другой. Доступа к этому классу через TypeDescriptor объекта.

    Image

    Абстрактный базовый класс, который предоставляет функциональные возможности для Bitmap и Metafile производных классов.

    ImageAnimator

    Анимирует изображение нажимать на основе времени.

    ImageConverter

    ImageConverter — Это класс, который может использоваться для преобразования Image объектов из одного типа в другой. Доступа к этому классу через TypeDescriptor объекта.

    ImageFormatConverter

    ImageFormatConverter — Это класс, который может использоваться для преобразования ImageFormat объектов из одного типа в другой. Доступа к этому классу через TypeDescriptor объекта.

    Pen

    Определяет объект, используемый для рисования прямых линий и кривых. Этот класс не наследуется.

    Pens

    Перья для всех стандартных цветов. Этот класс не наследуется.

    PointConverter

    Преобразует Point объект из одного типа данных в другой.

    RectangleConverter

    Преобразует прямоугольники из одного типа данных в другой. Доступа к этому классу через TypeDescriptor.

    Region

    Описывает внутреннюю часть графической формы, состоящей из прямоугольников и контуров. Этот класс не наследуется.

    SizeConverter

    SizeConverter Класс используется для преобразования из одного типа данных в другой. Доступа к этому классу через TypeDescriptor объекта.

    SizeFConverter

    Преобразует SizeF объектов из одного типа в другой.

    SolidBrush

    Определяет кисть одного цвета. Кисти используются для заливки графических фигур, таких как прямоугольники, эллипсы, круги, многоугольники и пути. Этот класс не наследуется.

    StringFormat

    Инкапсулирует сведения о структуре текста (например выравнивание, ориентация и позиции табуляции), операции с отображением (такие как вставка многоточия и замена национальных цифр) и возможности OpenType. Этот класс не наследуется.

    SystemBrushes

    Каждое свойство SystemBrushes класса SolidBrush — это цвет элемента изображения Windows.

    SystemColors

    Каждое свойство SystemColors класса Color структуры, представляющий собой цвет Windows отображения элемента.

    SystemFonts

    Указывает, что элементы отображения шрифтов, используемых для отображения текста в Windows.

    SystemIcons

    Каждое свойство SystemIcons класса Icon объекта для всей системы значков Windows. Этот класс не наследуется.

    SystemPens

    Каждое свойство SystemPens класса Pen — это цвет элемента изображения Windows и шириной 1 пиксель.

    TextureBrush

    Каждое свойство TextureBrush класс Brush объекта, использующего изображение для заполнения внутренней части фигуры. Этот класс не наследуется.

    ToolboxBitmapAttribute

    Позволяет задать значок для представления элемента управления в контейнере, такие как конструктор форм Microsoft Visual Studio.

    Структуры

    СтруктураОписание
    CharacterRange

    Определяет диапазон позиций символов в строке.

    Color

    Представляет цвет ARGB (альфа-канал, красный, зеленый, синий).

    Point

    Представляет упорядоченную пару из целое число со знаком x и y координаты, определяющую точку на двумерной плоскости.

    PointF

    Представляет упорядоченную пару из чисел с плавающей запятой x и y координаты, определяющую точку на двумерной плоскости.

    Rectangle

    Содержит набор из четырех целых чисел, определяющих расположение и размер прямоугольника.

    RectangleF

    Хранит набор из четырех чисел с плавающей запятой, определяющих расположение и размер прямоугольника. Для расширения функций области используйте Region объекта.

    Size

    Сохраняет упорядоченную пару целых чисел, указывающих Height и Width.

    SizeF

    Сохраняет упорядоченную пару чисел с плавающей запятой, обычно ширину и высоту прямоугольника.

    интерфейсы

    ИнтерфейсОписание
    IDeviceContext

    Определяет методы получения и освобождения существующего дескриптора контекста устройства Windows.

    Делегирует

    ДелегатОписание
    Graphics.DrawImageAbort

    Предоставляет метод обратного вызова в случае, когда DrawImage метод должен преждевременно отменить выполнение и прекратить рисование изображения.

    Graphics.EnumerateMetafileProc

    Предоставляет метод обратного вызова для EnumerateMetafile метод.

    Image.GetThumbnailImageAbort

    Предоставляет метод обратного вызова, позволяющий определить, когда GetThumbnailImage метод должен преждевременно прекратить выполнение.

    Перечисления

    ПеречислениеОписание
    ContentAlignment

    Указывает способ выравнивания содержимого на поверхности рисования.

    CopyPixelOperation

    Определяет, как цвет источника в операции копирования точки объединяется с цветом назначения для получения окончательного цвета.

    FontStyle

    Указывает сведения о стиле, применяемый к тексту.

    GraphicsUnit

    Указывает единицу измерения для заданных данных.

    KnownColor

    Задает известные системные цвета.

    RotateFlipType

    Указывает, сколько изображение поворачивается и ось для зеркального отражения рисунка.

    StringAlignment

    Задает выравнивание строки текста относительно прямоугольника размещения.

    StringDigitSubstitute

    StringDigitSubstitute Перечисление указывает, как выполняется замена знаков в строке в зависимости от языкового стандарта или языка пользователя.

    StringFormatFlags

    Указывает отображаемое и сведения о макете текстовые строки.

    StringTrimming

    Указывает, как удалить знаки из строки, которая не полностью помещается в форму структуры.

    StringUnit

    Указывает единицы измерения для строки текста.

    Комментарии

    Graphics Класс содержит методы для рисования на устройстве отображения. Классы, такие как Rectangle и Point инкапсулируют элементы GDI +. Pen Класс используется для рисования прямых линий и кривых, при классы, производные от абстрактного класса Brush используется для заливки фигуры.

    Внимание!

    Классы в System.Drawing пространства имен не поддерживаются для использования в службе Windows или ASP.NET. Попытка использования этих классов из одного из этих типов приложений может привести к неожиданным проблемам, например уменьшается службы исключения производительности и во время выполнения. Поддерживаемые см Windows Imaging Components.
