    Комментарии

    Graphics Класс предоставляет методы для рисования объектов на устройстве отображения. A Graphics связан с конкретным контекстом устройства.

    Вы можете получить Graphics путем вызова метода Control.CreateGraphics метод на объекте, который наследует от System.Windows.Forms.Control, или путем обработки элемента управления Control.Paint событий и доступ к Graphics свойство System.Windows.Forms.PaintEventArgs класса. Можно также создать Graphics объекта из образа с помощью FromImage метод. Дополнительные сведения о создании Graphics объекта см. в разделе Практическое руководство. Создание объектов Graphics для рисования.

    Можно нарисовать с помощью множества различных фигур и линий Graphics объекта. Дополнительные сведения о том, как рисование линий и фигур см DrawGraphicalElement метод для линии или фигуры, для рисования. Эти методы включают в себя DrawLine, DrawArc, DrawClosedCurve, DrawPolygon, и DrawRectangle. Дополнительные сведения о том, как рисование линий и фигур см. в разделе Рисование линий и фигур с помощью пера и Использование кисти для заливки фигур.

    Также можно рисовать изображения и значки, используя DrawImage и DrawIcon методов, соответственно. Чтобы выполнить перемещение набора битов данных о цвете с экрана на поверхность рисования объекта Graphics объекта см. в разделе CopyFromScreen. Дополнительные сведения о рисовании изображения с Graphics объекта см. в разделе Работа с растровыми и векторными изображениями.

    Кроме того, можно управлять система координат, используемая Graphics объекта. Дополнительные сведения о системе координат и манипулировать ими в разделе Системы координат и преобразования.

    Примеры

    В следующем примере кода предназначен для работы с Windows Forms и требует PaintEventArgs объекта. PaintEventArgs Объект называется e и является параметром Paint обработчика событий. Код выполняет следующие действия.

    • Создает рисунок из файла в формате JPEG. Файл называется SampImag.jpg и находится в папке примера.

    • Создает точку, в которой будет размещаться верхний левый угол изображения.

    • Отображает немасштабированное изображение на экране с помощью Graphics объекта.

    C#
    C++
    VB
    Копировать 
    private void DrawImagePointF(PaintEventArgs e)
{

    // Create image.
    Image newImage = Image.FromFile("SampImag.jpg");

    // Create point for upper-left corner of image.
    PointF ulCorner = new PointF(100.0F, 100.0F);

    // Draw image to screen.
    e.Graphics.DrawImage(newImage, ulCorner);
}
